# 🏥 Clinic Booking Automation System

A full automation system for managing clinic bookings using n8n, Supabase, and Telegram Bot.

---

## 🚀 Features

- ✅ Prevents double booking
- 🧠 Smart time slot suggestions
- 🔔 Automated notifications (confirmation & reminders)
- 🔄 Rescheduling and cancellation handling
- 👨‍⚕️ Multi-doctor support
- 📊 Organized booking data storage

---

## ⚙️ Tech Stack

- n8n (Workflow Automation)
- Supabase (Database)
- Telegram Bot API

---

## 🧠 How It Works

1. User selects service
2. Chooses doctor
3. Picks date & time
4. System checks availability
5. If occupied → suggests nearest available slot
6. If available → confirms booking instantly

---

## 🎯 Problem Solved

Manual booking systems cause:
- Double bookings ❌
- Missed appointments ❌
- Poor organization ❌

This system automates everything and improves efficiency.
